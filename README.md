## Stack

- Gatsby.js
- Contentful 
- Netlify

## Usage

### Dev

- Clone the repo.
- Run `npm install`.
- Create a file, `.env` in the root directory. 
- Define your `CONTENTFUL_ACCESS_TOKEN` in `.env`. 
- Run `gatsby develop.`

### Prod
- Clone the repo. 
- Define your `CONTENTFUL_ACCESS_TOKEN` on Netlify. 
- Build it.
